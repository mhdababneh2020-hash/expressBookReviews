# Express Book Reviews - REST API

A RESTful API for managing and reviewing books, built with Node.js and Express.

## Tech Stack

- Backend: Node.js + Express
- Authentication: JWT (JSON Web Tokens)
- HTTP Client: Axios
- Style: Async/Await + Promises

## Features

- Get list of all books
- Search books by ISBN, author, or title
- User registration and login
- Add and modify book reviews (authenticated users only)
- Delete your own reviews

## Run Locally

1. Clone the repo
2. Run: npm install
3. Run: node index.js
